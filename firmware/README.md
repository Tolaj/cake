# Firmware (planned)

Not yet implemented in this repository. Per the original MVP roadmap
([`../docs/CAKE_MVP_Roadmap.pdf`](../docs/CAKE_MVP_Roadmap.pdf)), this is where ESP12F/ESP32
firmware should live:

- MCU firmware for the [`esp12f-mcu`](../hardware/esp12f-mcu/) relay board — shift-register
  driven relay/switch IO, Wi-Fi provisioning, and an MQTT client.
- ESP32-based smart-home-hub firmware with Wi-Fi extender/mesh capability and presence
  detection (ESP-CSI), as described in the pitch deck.
