# Software (planned)

Not yet implemented in this repository. Per the original MVP roadmap
([`../docs/CAKE_MVP_Roadmap.pdf`](../docs/CAKE_MVP_Roadmap.pdf)), this is where the backend
and clients should live:

- **Server** — Node.js backend, with an [EMQX](https://www.emqx.io/) MQTT broker for
  device communication.
- **Dashboard** — React web dashboard for device control, status monitoring, automation/
  scheduling, and energy-usage data (see feature list in
  [`../docs/KIT-IRF_Pitch_Deck_Minisense.pdf`](../docs/KIT-IRF_Pitch_Deck_Minisense.pdf)).
- **Mobile app** — React Native app mirroring the dashboard's remote-control and
  scheduling features.
